# turkish-sensitive-text-redaction

Turkish Sensitive Data Redaction Tool

A lightweight PII redaction tool for Turkish text using a Named Entity Recognition (NER) token classification model.

Model used:
akdeniz27/bert-base-turkish-cased-ner (Hugging Face)

# Features

Turkish language support

BERT-based token classification (NER)

Redacts names, organizations, IBANs, currencies and numbers


Required Packages
pip install torch transformers
